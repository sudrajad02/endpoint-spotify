# endpoint-spotify
Endpoint Integrate With Spotify use expressJs

```bash
# install dependencies
$ pnpm install
or
$ npm install

# setting env
$ edit env.example
$ change name .env

# running app
$ pnpm start
or
$ npm start
```

All documentation endpoints can be accessed via the following link: [postman](https://documenter.getpostman.com/view/7838991/2s946eBDpZ)
